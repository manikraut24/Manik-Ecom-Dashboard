# Manik-Ecom-Dashboard
## E-Shop Sales Analytics 
Analyzed e-commerce sales data to identify customer behavior, product performance, and revenue trends. Built interactive dashboards to support data-driven decision making.

<img width="1165" height="656" alt="Dashboard_Image" src="https://github.com/user-attachments/assets/ce1ed01c-7c58-4521-acf8-1ab62060d187" />

## 🔹 Key Responsibilities

- Cleaned and transformed raw sales data using **Microsoft Excel** and **Power Query**
- Built an interactive dashboard in **Power BI** with dynamic filters (Quarter, Category)
- Created key KPIs including **Total Revenue (438K), Profit (37K), Quantity (5615), and AOV (121K)**
- Analyzed **state-wise sales performance** (Maharashtra, Madhya Pradesh, Uttar Pradesh, etc.)
- Performed **category-wise analysis** (Clothing 63%, Electronics 21%, Furniture 17%)
- Identified **monthly profit trends**, highlighting loss-making months
- Analyzed **payment mode distribution** (COD 44%, UPI 21%, etc.)
- Evaluated **top-performing sub-categories** (Printers, Bookcases)

Power BI (Dashboard, DAX basics, Data Modeling) </br>
Excel (Data Cleaning, Pivot Tables) </br>
Data Analysis, Data Visualization
